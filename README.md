# 文件密传 - 端到端加密文件中转站

一个轻量级的安全文件传输应用，支持文件加密上传、提取码分享和大文件分块传输，Nodejs跨平台一键离线部署。

## ✨ 功能特性

- 🔒 **端到端加密** - 使用AES算法对文件进行加密传输，密码由用户提供
- 🔗 **一键式分享** - 生成分享信息二维码，扫描即可获得下载地址及提取码
- 📦 **大文件支持** - 采用分片上传和下载技术，确保大文件可靠传输
- 🌐 **轻量跨平台** - Nodejs后端纯js且无数据库依赖，前端无框架纯原生实现
- 📱 **兼容性优化** - 兼容Chrome_56或Firefox_65以上版本的浏览器

## 🚀 快速开始

**开发环境：** 直接启动服务

```bash
npm start dev
```

**生产环境：** 打包后部署运行

```bash
# 打包项目文件
npm start build

# 打包后的产物可直接部署到其他服务器中
cd dist && node index.js
```

## 🔧 SSL 证书配置

使用**openssl** 配合 `certfile.cfg` 一键配置SSL证书可支持https访问，开启大文件后台下载能力，命令示例：

```bash
openssl req -x509 -newkey rsa:2048 -keyout private.key -out ./public/certfile.crt -days 3650 -config certfile.cfg -nodes -extensions v3_server
```

## 📖 使用指南

### 上传文件

1. 打开应用首页
2. 拖拽文件到上传区域或点击选择文件
3. 设置文件加密密码
4. (可选) 设置过期时间
5. 点击「加密上传」
6. 上传完成后，扫描二维码分享

### 下载文件

1. 切换到「下载文件」标签
2. 输入文件提取码
3. 输入文件解密密码
4. 点击「下载并解密」开始下载

---

**注意**：

1.首次运行前请确保 `uploads/` 和 `data/` 具有写权限；

2.本应用未接入数据库服务，建议只在内部范围轻量使用。
